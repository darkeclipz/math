# Single variable derivatives

## Formal definition

Let $f(x)$ be a function with a single variable $x$, then the formal definition of the derivative is:

$$
\dfrac{\mathrm{d}f}{\mathrm{d}x} = \lim_{h \rightarrow \infty} \dfrac{f(x+h) - f(x)}{h}.
$$
